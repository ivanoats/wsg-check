You do **not** need to sign a separate document. By opening a pull request or submitting a patch to this repository you are indicating your agreement to this CLA.

Include the following sign-off line in **every commit message** in your pull request:

```

```

You can add this automatically with:

```bash
git commit -s -m "feat: my contribution"
```
